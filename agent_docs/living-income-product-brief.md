# Product Brief: Living Income Data Pipeline

## Overview
A data pipeline script to ingest Living Income documents into the Akvo RAG Knowledge Base. This script will handle local file processing and bulk upload/processing via the RAG API.

## Objectives
- Create a reusable Python script `kb_init_living_income.py`.
- Support local document ingestion from `./script/downloads/living_income`.
- Integrate with the existing RAG API utility.
- Allow configuration of target system (URL/Auth).

## Target Audience
Internal developers and data managers re-uploading knowledge bases for Akvo RAG.

## Timeline
Phase 1-8 to be completed in one sprint.
